# Discover Command

## Description

Collect and summarize key information from multiple sources to provide actionable insights.

## Trigger

Use when exploring a new topic, project, or unfamiliar domain requiring quick context.

## Steps

1. Identify the main topic or keywords.
2. Gather relevant data from internal and external sources.
3. Summarize findings into clear, structured points.
4. Highlight patterns, gaps, and next steps.

## Output Format

- **Summary:** Brief overview of findings.
- **Key Points:** Bullet list of critical insights.
- **Next Steps:** Suggested actions or areas for deeper research.

## Examples (Optional)

**Input:** "Discover insights on AI-driven testing tools"  
**Output:**

- Summary: AI tools improve test coverage and reduce manual effort.
- Key Points: Leading tools include X, Y; common features are automation and analytics.
