# Deep Think Command

## Description

Force deep, sequential reasoning for complex problems using the `sequentialthinking` tool.

## Trigger

User asks to "think deeply", "analyze carefully", or use "deep think".

## Steps

1. **Engage Tool**: Always call the `sequentialthinking` MCP.
2. **Break Down**: Deconstruct the task into clear, step-by-step reasoning.
3. **Validate**: Only proceed to the next step once the previous one is fully analyzed.
4. **Prioritize**: Focus on logical consistency and thoroughness over speed.

## Output Format

Output from the `sequentialthinking` tool (step-by-step analysis).
