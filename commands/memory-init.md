# Init Memory Bank Command

## Description

Initialize the Memory Bank structure for the project to track context and decisions.

## Trigger

User runs `init memory bank`.

## Steps

1. **Create Directory**: Ensure `.cursor/rules/memory-bank/` exists.
2. **Create Core Files**:
    - `projectbrief.md`: Scope, goals.
    - `productContext.md`: Purpose, UX goals.
    - `activeContext.md`: Current focus, next steps.
    - `systemPatterns.md`: Architecture, patterns.
    - `techContext.md`: Stack, constraints.
    - `progress.md`: Status, known issues.
3. **Populate**: Fill with initial templates or project-specific data if available.

## Output Format

Confirmation message indicating the Memory Bank has been initialized.
